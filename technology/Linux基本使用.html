    # Linux基本使用

    你现在已经了解了Linux文件结构了

    下面来说说linux的常用命令 (我自己常用的) 吧

    # 一.文件列表

    Linux下一切皆文件

    下列是文件列表的操作

    命令 说明
    ls 当前目录所含文件及子目录
    ls -a 列出目录下的所有文件，包括以 . 开头的隐含文件
    ls -l 列出文件的详细信息
    ls -al 显示当前目录下的所有文件及文件夹包括隐藏的.和…等的详细信息
    pwd 显示当前所在工作目录的全路径

    # 二.操作目录

    目录相当于windows下的文件夹

    命令 说明
    cd+目录名/ 切换当前目录为其他目录(tab键补全)
    cd .. 退回上一级目录
    mkdir +目录 创建目录
    mkdir -p 目录/子目录 创建目录/子目录
    rmdir+目录 删除目录

    # 三.操作文件

    命令 说明
    touch+文件名 创建一个新文件
    cat+文件名 查看文件
    rm+文件名 删除文件(删前询问)y确认n取消
    cp 将一个文件复制成另一个文件或复制到某目录时，需要指定源文件名与目标文件名或目录,如cp a.txt b.txt 将a.txt复制为b.txt文件
    mv 移动或重命名 mv+文件名+目录 为移动,mv+文件名+文件名 为重命名
    find 指令用于查找符合条件的文件,如find / –user mack –ls表示查找用户mack的文件
    grep 查找符合条件的字符串,如grep lang a.txt 在文件中查找lang

    # 四.Vim编辑

    在Linux下一般使用Vi编辑器来编辑文件

    它是Linux和Unix系统上最基本的文本编辑器

    相当于windows系统自带的记事本

    Vim(Vi improved)是vi编辑器的加强版，比vi更容易使用

    vim既可以查看文件也可以编辑文件

    有些vim爱好者使用起来特别顺手

    故用来代替常见ide敲代码

    然而作者并不习惯

    所以只说一下基本使用(对配置文件修改)

    # 1.Vim四种模式

    Vim上存在四种模式每种模式只能对应一种操作

    • 命令模式 : 输入操作指令,如复制粘贴,撤销恢复等

    • 编辑模式 : 对文件内容进行修改

    • 底行模式 : 文件保存,退出,查找替换等

    • 可视化模式(不常使用)

    # 2.基本语法

    打开文件

    vim 文件名称
    1
    • 如果文件存在直接打开
    • 文件不存在则会在内存中创建一个新文件

    打开文件以后,默认进入命令模式

    如果处于处于其他模式,连按两次ESC会返回命令模式

    进入命令模式后,可以先光标定位(可选),然后进入编辑模式操作

    下面是进入编辑模式的常用命令

    命令 说明
    i 表示在当前位置前插入
    I 表示在当前行插
    a 表示在当前位置后插入
    A 表示在当前行尾部插入
    o 表示在当前行之后插入一行
    O 表示在当前行之前插入一行

    编辑好文件之后需要保存

    需要经过命令模式再到底行模式(按ESC)

    在切换到底行模式:

    shift :
    1

    底行模式常用操作

    命令 操作
    q! 不保存退出
    wq 保存并退出

    至于其他操作,用到的时候再去搜 以后更新shell编程
