    # 一.Lambda

    Lambda是JDK8的新特性语法糖

    是函数式编程的重要思想

    关注于对数据进行了什么操作

    选中小括号中的类按alt+回车快速生成lambda表达式

    # 1.基本格式

    (参数列表)->{}
    1

    # 2.使用条件

    1. 使用Lambda必须具有接口，且要求接口中有且仅有一个抽象方法。无论是JDK内置的Runnable ,Comparator 接口还是自定义的接口，只有当接口中的抽象方法存在且唯一时，才可以使用Lambda。
    2. 使用Lambda必须具有上下文推断。也就是方法的参数或局部变量类型必须为Lambda对应的接口类型，才能使用Lambda作为该接口的实例。

    # 3.示例:

    创建启动线程:

    new Thread(new Runble(){
    @Override
    public void run(){
        System.out.println();
    }
}).start();

    public class Hero {
    private String id;
    private String name;
    private String identity;

    public Hero(){}

    public Hero(String id, String name, String identity) {
        this.id = id;
        this.name = name;
        this.identity = identity;
    }

    public String getId() {
        return id;
    }

    public void setId(String id) {
        this.id = id;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public String getIdentity() {
        return identity;
    }

    public void setIdentity(String identity) {
        this.identity = identity;
    }

    @Override
    public boolean equals(Object o) {
        if (this == o) return true;
        if (o == null || getClass() != o.getClass()) return false;
        Hero hero = (Hero) o;
        return Objects.equals(name, hero.name);
    }

    @Override
    public int hashCode() {
        return Objects.hash(id, name, identity);
    }

    @Override
    public String toString() {
        return "Hero{" +
                "id='" + id + '\'' +
                ", name='" + name + '\'' +
                ", identity='" + identity + '\'' +
                '}';
    }
}

    # 三.Optional

    编写代码时经常出现空指针异常

    所以在很多情况下我们需要做各种非空的判断

    过多的判断会让我们的代码显得臃肿不堪

    JDK8中引入了Optional

    可以把我们具体的数据封装到Optional对象内部,再使用封装好的方法操作封装进去的数据就可以非常优雅的避免空指针异常

    # 1.创建对象

    class Solution {
    public static void main(String[] args) {
        Hero hero = getHero();
        Optional<Hero> optionalHero = Optional.ofNullable(hero);
        optionalHero.ifPresent(hero1 -> System.out.println(hero1));
    }

    public static Hero getHero(){

        return null;
    }
}

    # 3.安全地获取值

    使用orElseget

    如果值为空,返回你设置的新值

    不为空则返回原值

    class Solution {
    public static void main(String[] args) {
        Hero hero = getHero();
        Optional<Hero> optionalHero = Optional.ofNullable(hero);
        Hero hero2 = optionalHero.orElseGet(new Supplier<Hero>() {
            @Override
            public Hero get() {
                return new Hero();
            }
        });
        optionalHero.ifPresent(hero1 -> System.out.println(hero1));
    }

    public static Hero getHero(){
        Hero hero = new Hero("001", "刘备", "蜀");
        return null;
    }
}
